# 🧩 tiny-toys

✨ 一个存放 `AI` 辅助及自主编写的小型玩具级代码的仓库，记录技术练习与创意实现 ✨

## 📖 项目简介

本仓库用于归档本人开发的各类 `tiny toys`，即轻量、简洁、用途明确的小型代码模块，其中部分代码借助 `AI` 工具辅助编写、优化，部分为自主开发的技术练习或实用小工具，主打轻量化、易读、可复用，无需复杂依赖。

项目核心目标：

- 记录个人编程学习与技术积累过程，沉淀各类小型功能的简洁实现方案，方便后续查阅回溯；
- 探索 `AI` 辅助编程的实用场景，借助 `AI` 提升开发效率、优化代码逻辑与质量，沉淀协同编码经验；
- 分享轻量、可复用的小型代码模块，为有类似需求的开发者提供参考，减少重复造轮子成本；
- 保持代码的简洁性与可读性，便于后续查阅、修改和扩展，适配碎片化开发与练习场景。
