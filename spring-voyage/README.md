# 春天熊の知行录

一个融合了技术极客精神与文艺浪漫气息的个人网站。白天构建代码帝国，深夜挥洒海明威式文风，记录从“武理四年”到职场深耕的代码探索、算法沉淀以及生活中的诗与远方。

在线访问：[https://springbear.cn](https://springbear.cn)

## 功能特点
- 💻 技术沉淀：涵盖 `Go`、`Java`、`C语言`及前端等多技术栈的学习笔记、系统方案与实战经历总结
- 🎯 算法进阶：收录 `300+` 篇力扣（LeetCode）等平台的解题历程，图文并茂夯实编程内功基石
- ✒️ 程序人生：用笔尖勾画周末的诗意，分享程序员的日常感悟、闲庭碎笔与青春足迹
- 📚 精神自留地：特设读书札记与电子文藏，在《代码整洁之道》与《百年孤独》间自由穿梭
- 🌟 浪漫彩蛋：独创 `git commit` 浪漫文风，展现“写代码如写情书”的极致个人特色

## 阅读指南
1. 直接访问主页即可浏览最新博客动态，感受“活力如春，坚毅如熊”的极客氛围
2. 通过顶部导航栏的「分类」或「标签」，可快速检索特定的技术干货或生活随笔

## 注意事项
- 站点页面已进行响应式适配，但为了代码块的最佳阅读体验，建议使用 PC 端浏览器访问
- 站点内置的部分前端小工具依赖浏览器 `localStorage`，清空缓存会丢失本地数据
